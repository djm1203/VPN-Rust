# Classification Footer Templates

Use the appropriate footer at the bottom of every document based on its classification level.

## Restricted

```
RESTRICTED — Internal use only. Principal approval required for distribution.
```

## Confidential

```
CONFIDENTIAL — Authorized recipients under NDA only.
```

## Public

```
PUBLIC — Approved for general audience distribution.
```
