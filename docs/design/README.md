# Design — project

UI/UX mockups, wireframes, design system tokens, and visual specifications.

Each feature's design output lives in a `B-{NNN}-{slug}/` subdirectory with a `MANIFEST.md`.

## Convention

- Directory pattern: `B-{NNN}-{kebab-slug}/`
- Each directory contains a `MANIFEST.md` with backlog ID, title, file listing
- Design docs are produced in dual format: `.md` (structural) + `.html` (visual, PPF dark theme)
